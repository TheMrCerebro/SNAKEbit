[*] Introduction:
=============

 OMG, the snake is hungry, help her!!. Guide the snake into the box and make it
 eat all the eggs you find. It is very simple, the more the snake eats, the longer
 it becomes.


=========
[*] Controls:
=========

 Up, Down, Left, Right - Move de snake
 X - Accept
